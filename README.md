#  API Portfolio Project

This project is a simple and modern portfolio-style website created using **HTML, CSS, and JavaScript**.
It displays my **Skills** and **Projects** using clean UI cards, dark mode, search functionality, and smooth animations.

---

##  Features

*  **Dark / Light Mode Toggle**
*  **Search Bar** (Filter skills & projects)
*  **Modern UI Cards**
*  **Smooth Animations**
*  **Fully Responsive**
*  **Custom Data Display (No API needed)**

##  How to Run

1. Open the project folder in VS Code
2. Install the **Live Server** extension
3. Right-click on `index.html`
4. Click **Open with Live Server**

Your portfolio will open in the browser.
